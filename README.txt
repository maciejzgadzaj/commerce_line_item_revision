
DESCRIPTION
===========

Module features:

  * adds revision support to Commerce Line Item entity type,

  * alters admin Order settings page adding an option to "Create
    new revisions when line items are updated by default",

  * provides Views integration for line item revisions,

  * provides a basic view listing revisions of a specific line item,

  * alters admin Line items view displayed on admin order details page,
    re-enabling the "Line item ID" column linked to the Line item revisions
    page.



INSTALLATION
============

1) Enable the module.

2) Enable "Create new revisions when line items are updated by default" option
   on the admin Order settings page (admin/commerce/config/order) if required.
